CNPJ360 API

API RESTful para consulta de CNPJ e dados de empresas

Base URL: https://kidion.online/cnpj360/api/v1/

Autenticação

Todas as requisições devem incluir sua API Key no header:

X-API-Key: sua_api_key_aqui

Endpoints

GET /cnpj/{cnpj}

Consulta CNPJ em tempo real na Receita Federal

Exemplo: 
GET /api/v1/cnpj/12345678000199
GET /empresa/{cnpj}

Busca empresa no banco local

POST /empresa/busca

Busca avançada de empresas

Body: 
{
  "cidade": "São Paulo",
  "uf": "SP",
  "porte": "Micro Empresa",
  "limit": 20,
  "offset": 0
}
GET /empresa/socios/{cnpj}

Lista sócios da empresa

GET /empresa/cnaes/{cnpj}

Lista CNAEs da empresa

GET /estatisticas

Estatísticas do sistema

Respostas

Todas as respostas seguem o formato:

{
  "status": "success",
  "code": 200,
  "timestamp": "2024-01-15T10:30:00-03:00",
  "api_version": "v1",
  "data": { ... }
}

Códigos de Erro

  • 200 - Sucesso
  • 400 - Requisição inválida
  • 401 - API Key inválida
  • 404 - Recurso não encontrado
  • 429 - Limite de requisições excedido
  • 500 - Erro interno do servidor
